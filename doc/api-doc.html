Présentation de l'API

Bienvenue sur l'API de Krysto IO, dédiée au développement durable et à la gestion de l'empreinte écologique. Consultez les sections ci-contre pour plus d'informations sur les fonctionnalités disponibles.

Authentification

Cette API utilise une authentification basée sur des tokens pour sécuriser les accès. Vous devez inclure un token valide dans chaque requête.

Endpoints

Gestion des erreurs

Les erreurs sont renvoyées sous forme de réponses JSON contenant le code et le message d'erreur pour aider au débogage.
